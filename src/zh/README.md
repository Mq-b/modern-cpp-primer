# Modern C++ Primer

## 介绍

本项目是一个以认知阶段为主线的现代 C++ 教程，为此有三个主要目标：
* 内容准确，尽量避免歧义
* 跟随 C++ 标准
* 内容由浅入深，减少学习负担
